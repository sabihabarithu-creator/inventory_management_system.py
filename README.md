# 📦 Inventory Management System

A simple Inventory Management System built using Python. This project allows users to manage products by adding, viewing, updating, searching, and deleting inventory items through a menu-driven interface.

## 🚀 Features

- Add a new product
- View all products
- Update product quantity
- Delete a product
- Search for a product
- Easy menu-driven interface

## 🛠 Technologies Used

- Python 3

## 📂 Project Structure

```
inventory-management-system/
│── inventory_management_system.py
│── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the following command:

```bash
python inventory_management_system.py
```
